# モード再編とメニュー整理 設計書（P10）

散在していた機能を **3つのモード（学習／パズル／自由）** に整理し、各モードで使う道具だけを見せて
メニューをわかりやすくする。あわせて不斉炭素マークの扱いを整理する。
2026-07-23 ユーザー合意（3モード・自由=完全な白紙の砂場・不斉は判定オプションに統合・設計書先行）に基づく。
実装前の合意用ドキュメント。実装の進行に合わせて更新する。

---

## 1. ねらい

- 現状、パズル用UI・反応機構ビュー・横断的な道具（クイズ／命名／立体／異性体／官能基まとめ／反応実行／不斉マーク）が
  同じ画面に併存し、初見では何をする画面か分かりにくい。
- **「今どのモードか」を明示し、そのモードで使う道具だけを出す**ことで、迷いを減らす。
- 既存機能は削除しない。配置とモード帰属を整理するだけ（挙動・判定ロジックは不変）。

## 2. 3つのモード

| モード | 目的 | 特徴 |
|---|---|---|
| 🧩 **パズル** | お題の分子を組んで正誤判定 | ステージ選択・お手本・判定ボタン。**正解を当てる**体験 |
| 📚 **学習** | 反応や仕組みを見て学ぶ | 反応機構ビューア・チュートリアル・クイズ（同じ/違う・命名）。**受け身で学ぶ／腕試し**体験 |
| 🧪 **自由** | 白紙から自由に組んで調べる | お題・判定なしの砂場。作図＋反応実行＋分析ツール。**手を動かして試す**体験 |

- 上部（ヘッダー下）に **モード切替タブ**（🧩パズル / 📚学習 / 🧪自由）を置く。
- 左パレット（TOOLS: 操作モード・結合タイプ・原子・モジュール）は **3モード共通**で常に表示（作図は全モードで行う）。
- 右パネルの内容だけがモードで切り替わる。

## 3. モード別のUI（右パネル）

### 3.1 🧩 パズルモード
- シリーズ／問題の選択、お題カード（名称・化学式・説明・お手本表示）
- **判定ボタン**と結果表示
- **判定オプション**（新設の折りたたみ枠）:
  - ☑ 不斉炭素（*）の指定も判定する ← 不斉判定の ON/OFF スイッチ（#4）
    - ONにすると、判定時に不斉炭素マークの正誤も採点する（現行 asymmetricMode の判定を明示化）
- 「いま描いている分子」カード（名称・化学式のライブ表示）は補助として表示

### 3.2 📚 学習モード
- **反応機構ビューア**（機構選択・ステップ送り・再生・生成物予測）
- **クイズ**: 同じ化合物？クイズ・命名クイズ
- **チュートリアル**（❓ヘルプは全モード共通でヘッダーに残す）
- 作図キャンバスは共有（機構の予測モードで分子を組むため）

### 3.3 🧪 自由モード（完全な白紙の砂場）
- **お題・正解判定・ステージ選択は一切出さない**
- 「いま描いている分子」カード（名称・化学式）
- **⚗ この分子の反応**（プロパティ表示・反応実行・名称呼び出し）
- **分析ツール**: 📚異性体を調べる・🧊立体で見る・🔤官能基をまとめる
- 名称からの分子呼び出しで、既存の化合物を出して反応・分析を試せる

## 4. 不斉炭素マークの扱い（#3・#4）

- **マーク用ボタンを左パレットへ移設**（#3）: 「操作モード」枠に *不斉炭素マーク* をトグルボタンとして置く
  （Select/Bond/Erase と並ぶ第4の操作モード的な扱い）。道具として自然な位置にする。
- **判定に含めるかのスイッチ**（#4）: パズルモードの「判定オプション」に
  「☑ 不斉炭素の指定も判定する」を置く。マーク作業（左パレット）と、判定に反映するか（右の判定オプション）を分離する。
- 現行の `asymmetricMode`（マーク編集モード）と、判定時のマーク照合は、内部的には維持しつつUIを上記に再配置する。
- 自由モードでは判定が無いので、不斉マークは「立体で見る」的な確認用途として使える（判定はしない）。

## 5. モード切替の挙動

- モード切替時、作図中の分子は保持する（消さない）。右パネルの表示だけ切り替える。
- パズル→他モードに移ると判定結果表示はクリア。学習の反応機構モードから出るときは reactionPlayer.exit()。
- 既定モードは 🧩パズル（初回訪問時。従来どおりお題が出る）。
- モード状態は localStorage に保存し、次回同じモードで開く（任意）。

## 6. 実装マイルストーン

- **M1**: モード切替タブの新設と、右パネルの3分割（既存要素をモードごとの `<section>` に振り分けて表示/非表示）。挙動は現状維持。
- **M2**: 不斉マークボタンを左パレットへ移設＋パズルの「判定オプション」に不斉判定スイッチを新設（#3・#4）。
- **M3**: 自由モードの純化（お題・判定の非表示、分析ツールの集約）と、モード保存。
- **M4**: 各モードのチュートリアル導線の見直し（FAQに「モードとは」を追加）。

## 7. リスク・未決事項

- **回帰テスト**: モードで要素の表示/非表示が変わるため、テストは対象モードに切り替えてから要素を操作する必要がある。
  テストヘルパーに `setMode(name)` を追加し、既存テストの前提（要素が常に見えている）を壊さないよう調整する。
- **反応機構モードとの関係**: 現行の「反応機構モード」チェックボックスは、学習モード内の一機能に格下げする（トグルは残す）。
- **命名クイズ・同じ/違うクイズ**: モーダルなので、学習モードのボタンから開く形に。モード外からは出さない。
- **自由モードの初期状態**: 空キャンバス。名称呼び出しの導線を目立たせる。
- **判定オプションの拡張余地**: 将来 cis/trans 判定の厳密化や R/S を入れる場合も、この「判定オプション」枠に追加できる形にしておく。
